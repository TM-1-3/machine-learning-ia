
# Dependecies

- Python 3.12 or above

# Prepare your environment 

To prepare your environment execute the following commands:

```bash
#create a python environment
python3 -m venv venv 

#activate the environment
source venv/bin/activate

#install packages 
pip install -r requirements.txt
```

# How to run this program? 

```bash
python3 learn.py
streamlit run app.py
```

The first command will train a custom model with the necessary data and the second one will open the web app so that we can test the probability for our own values.
